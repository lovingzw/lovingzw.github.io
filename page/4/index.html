发表于 2018-04-19 更新于 2018-05-23 分类于 external
本文字数： 2.2k

不用介绍什么是消费主义了吧

消费主义对我们生活侵蚀之广之深，每个身处工业化和城市化浪潮的人都应该有所体会。商品的制造被标准化后，对商品的消费也同时被标准化。何种消费应该对应哪种场景、哪类人群，又或者是哪种程度努力后的奖励，这样的等式早已植根在社会的土壤之中。而消费主义不仅存在于物质层面，在社会舆论上，对热点新闻和人物的过度挖掘报道早已让人担忧。不仅是人的劳动变成了商品，人本身也被当作「商品」，接受无法超越的异化，被摆上流水线制造成商品供他人观赏咀嚼。

后现代的出路绝不是小清新

发表于 2018-04-10 更新于 2018-05-24 分类于 external
本文字数： 856

不鸟万如一在 《一天世界》 里曾经多次讨论过「小清新」式观念现在所面临的困境。

小清新是舒适的

面对低俗但火的东西，比如今日头条，比如快手，小清新们是不会让它们真正进入自己的世界的。于是，他们的选择只有两种：要么洁身自好，将其完全隔绝；要么就用其存在的合理性来包容它，用一种温和的冷暴力处理。带有强度的事物，小清新们总是避之不及，任何事情都「刚刚好」就最好了。一如「做一个有趣的人」这句小清新的 cliché，消解了现实生活的强度。而且重要的是，是否有趣难以被传统的价值体系评判，成为有趣的人不需要强度，真是毫不费力。
