封闭工具 -> 封闭心态

发表于 2018-04-28 更新于 2018-05-24 分类于 external
本文字数： 490

使用封闭的工具是无法获得开放心态的。

众所周知的微信闭环将越来越多人圈在微信的笼子里，因为几乎所有需求都能在微信这个操作系统里得到解决。于是，在使用微信这个工具的同时，有不少人已经将微信当作互联网的全部。

即使知道文章会被 404，也要在公众号发表，因为这是吸引注意力最快的方式。在封闭环境中无法接触其它途径，植入一种思维习惯，让公众号逐渐变成了发表意见的唯一方式。可悲的是大家已经养成了对 404 的放任态度。有些人已经不知道被在微信里被禁止访问的外链内容，其实是可以把链接复制到浏览器里访问的。

世界正变得越来越小。不是连接变得方便了，而是观念中的世界，真的在变小。

同样，微博也是封闭的。禁止外部对内容的访问，封闭的内部链接，这都已经是社交网络的常态了。除了封闭环境，更值得警惕的是用户视角的封闭。在展现内容的时候，社交媒体都是一条单一的瀑布流。世界在使用者眼中的景观也顺流而下变得狭窄。单一的内容渠道并不拓展对世界的知觉，而只是在增强已有的认知。对偶像的迷恋和对异见者的愤怒随着瀑布流的刷新不断放大。网络暴力在情绪高涨下滋生，自我意识不断膨胀。

世界在变小，自我在膨胀。最后会怎样？
