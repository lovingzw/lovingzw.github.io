Sublime Text 3 package control 不显示的问题

发表于 2016-12-06 更新于 2018-07-22 分类于 skills
本文字数： 386

最近突然在 Sublime Text 3 中找不到 package control 了，按照官网的两种方法试图重新安装，都不显示。本来还想把 ST3 卸载重装，因为之前装了太多插件，主题的配置也花了很长时间，不想再折腾了。于是，在检查配置文件的时候，发现了问题所在。

在 ~/Application Support/Sublime Text 3/Packages/User/Preferences.sublime-settings 这个配置文件中，居然把 package control 禁用了。把 Package Control 那项删掉，保存重启 ST3 即可。

于是又可以愉快地玩耍了。
